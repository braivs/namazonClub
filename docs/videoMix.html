Смешанная борьба

В нашем клубе смешанная борьба - это СОРЕВНОВАТЕЛЬНАЯ борьба между мужчиной и женщиной (миксрестлинг), которая проводится по правилам САБМИШН ГРЭППЛИНГА без ограничения по времени каждой схватки.
